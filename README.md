### 说明
改造来自 bootstrap 教育培训侧使用的模板适配 Django, 没有逻辑处理, 模板来自于 http://www.bootstrapmb.com/item/9982/preview

### 使用
1、下载
```
git clone git@github.com:pinsily/django_template_education.git
```

2、安装依赖
```
pip install requirement.txt
```
